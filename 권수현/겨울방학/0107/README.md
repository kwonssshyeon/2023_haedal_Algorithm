### 다른 풀이

s와 *를 하나의 큐에두고 풀이를 할 수도 있을것 같다<br>

*를 먼저 큐에 넣고, s를 다음에 넣는다면 순서도 지켜질것 같다.<br>

다만, 매번 배열에서 *,s인지를 확인해야하기 때문에 그냥 (s인지 *인지,y좌표,x좌표) 이렇게 3-tuple로 큐에 넣는게 더 좋을 것 같다<br>
