x-y 범위내에서 같은 room이었으면 다같이 바꾸면 될꺼라고 처음에 생각했다.

근데 y밖에 있는것도 같은 room이었다면 함께 업데이트 해야한다...

코드 고치기 귀찮아서 억지로 풀었당..(나아아아중에 수정하셍)

```python
before = room[y]

j=1
while  (y+j)<=n and room[y+j]==before:
    room[y+j]=room[y]
    j+=1
```
