### 문제 설명

<p>예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.</p>

### 입력

 <p>첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.</p>

### 출력

 <p>첫째 줄부터 2×N-1번째 줄까지 차례대로 별을 출력한다.</p>

### 풀이

1. N을 입력받는다.
2. 입출력 예시를 보고 생각을 해 본다.
3. 공백과 별 사이의 관계를 잘 생각해보고 출력한다.
