[1068 트리](https://www.acmicpc.net/problem/1068)  
===========
트리에서 주어진 노드 삭제 후 리프노드 개수 출력  

-----------------------

#### 알고리즘  
1. 트리 초기화  
   부모 노드를 받아서 초기화해야 함. 특이함...  
   부모가 -1이면 내가 루트
   부모의 왼쪽 자식 자리부터 채우면 됨.
  
   노드 - 부모
   0 - -1
   1 - 0
   2 - 0
   3 - 1
   4 - 1  

2. 노드 삭제  
   삭제할 노드는 하나만 주어지는 듯?
   
4. 리프 개수 세기  
   자식 노드가 없으면 +1
